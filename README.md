# Library-Management-System
A complete Library Management System with Book and Member classes implementing proper OOP principles, encapsulation, and basic library operations.

Project Description

The Library Management System is a console-based Java application designed to manage library operations efficiently using Object-Oriented Programming (OOP) principles. The system allows librarians to manage books and members, perform borrowing and returning operations, and search for books based on various criteria.

✅ Goals

To design a simple and efficient library system using Java

To apply core OOP concepts such as encapsulation, abstraction, and modularity

To provide a user-friendly, menu-driven interface for library operations

✅ Objectives

Create separate classes for Book, Member, and Library

Maintain proper data encapsulation using private fields and public methods

Allow members to borrow and return books with availability validation

Enable search functionality by book title and author

Implement a menu-driven console interface for easy interaction

Ensure clean, readable, and maintainable code structure

**Prerequisites

Java Development Kit (JDK 8 or higher)

Any Java IDE (IntelliJ IDEA / Eclipse / NetBeans)
OR

Command Line (Terminal / Command Prompt)

Compile the Program

Open terminal in the project folder and run:

javac *.java

Run the Application
java LibrarySystem
